## 1. Getting Started

### Prerequisites

- **Apple Device**: iPhone 12+, iPad Pro, Mac (M1+), Apple Watch (optional)
- **HealthKit Access**: Grant permission for heart rate and HRV data
- **Headphones**: Quality headphones for Multidimensional Brainwave Entrainment (AirPods Pro/Max recommended)
- **Optional**: Ableton Push 3 for LED feedback, MIDI controller

### First Launch

1. Open Echoelmusic
2. Grant HealthKit permissions (required for biofeedback)
3. Grant Microphone access (for voice input)
4. Complete the onboarding tutorial
5. Put on your Apple Watch (if available)

### Initial Calibration

Before your first session, calibrate your baseline:

```
Settings → Biofeedback → Calibrate Baseline
```

This 3-minute session establishes your:
- Resting heart rate
- Baseline HRV
- Default breathing rate
- Coherence threshold

## 2. Creative Session: Biofeedback Setup

### Understanding Bio-Reactive Audio

Echoelmusic transforms your physiological state into music:

| Bio Signal | Audio Parameter | Effect |
|------------|-----------------|--------|
| Heart Rate | Tempo | Faster heart = faster tempo |
| HRV | Reverb/Space | Higher HRV = more ambient space |
| Coherence | Harmonic Complexity | High coherence = Fibonacci harmonics |
| Breathing | Modulation | Breath cycle = filter sweeps |

### Starting a Session

1. **Choose Session Type**:
   - **Meditation**: Calm, ambient, 50-70 BPM
   - **Focus**: Balanced, rhythmic, 80-100 BPM
   - **Energize**: Uplifting, dynamic, 110-130 BPM
   - **Free Create**: No constraints, full control

2. **Connect Biofeedback Sources**:
   ```
   Home → Connect → Select sources:
   ✓ Apple Watch (heart rate, HRV)
   ✓ HealthKit (historical data)
   ✓ Breath Sensor (optional)
   ✓ Camera (face tracking)
   ```

3. **Configure Bio-Mapping**:
   ```
   Settings → Bio-Mapping → Customize:
   - Sensitivity: 1-10 (how responsive)
   - Smoothing: 1-10 (how gradual)
   - Range Lock: ON/OFF (prevent extreme values)
   ```

### Real-Time Coherence Training

The app teaches you to achieve coherence:

1. **Watch the Coherence Ring**: Green = coherent, Red = stressed
2. **Follow the Breath Guide**: Inhale 4s → Hold 4s → Exhale 6s
3. **Listen for Audio Feedback**: Harmonious sounds indicate coherence
4. **Observe Visual Changes**: Mandala patterns bloom at high coherence

### Tips for Better Biofeedback

- **Consistent Environment**: Same time, same place
- **Minimize Distractions**: Silent notifications
- **Proper Watch Fit**: Snug but comfortable
- **Stay Hydrated**: Dehydration affects HRV
- **Avoid Caffeine**: 2 hours before session

## 3. Music Creation Workflow

### Capture Mode

Record your bio-reactive session:

1. **Start Recording**:
   ```
   Tap Record (●) → Session begins
   ```

2. **During Recording**:
   - Bio data is captured alongside audio
   - MIDI events are logged with bio-context
   - Visual state is synchronized

3. **End Recording**:
   ```
   Tap Stop (■) → Session saved
   ```

### AI-Assisted Composition

The AI composer suggests based on your bio-state:

1. **Melody Generation**:
   ```
   AI → Generate Melody
   Key: [Auto-detect from mood]
   Scale: [Suggested: Pentatonic for calm]
   Bars: [8-16 for development]
   ```

2. **Chord Progression**:
   ```
   AI → Suggest Chords
   Mood: [Detected: Meditative]
   Progression: I - vi - IV - V
   ```

3. **Drum Pattern**:
   ```
   AI → Generate Drums
   Style: [Bio-reactive: Calm]
   Output: Sparse, breath-synced hits
   ```

### Manual Editing

Refine AI suggestions:

1. **Piano Roll**: Edit MIDI notes
2. **Arrangement View**: Structure your piece
3. **Mixer**: Balance levels
4. **Effects Chain**: Add processing

### Gesture Control

Use hand gestures for expressive control:

| Gesture | Control |
|---------|---------|
| Open Palm | Sustain/Hold |
| Fist | Mute/Cut |
| Pinch | Filter Close |
| Spread | Filter Open |
| Rotation | Pan/Modulation |

### Face Tracking

Facial expressions map to audio:

| Expression | Parameter |
|------------|-----------|
| Smile | Brightness/High frequencies |
| Frown | Darkness/Low frequencies |
| Eyebrow Raise | Pitch Bend Up |

## 4. Production & Mixing

### Spatial Audio Setup

Configure 3D/4D audio:

1. **Field Geometry**:
   ```
   Spatial → Field Type:
   - Fibonacci Spiral (high coherence)
   - Grid (structured)
   - Sphere (immersive)
   ```

2. **Source Placement**:
   ```
   Each voice has:
   - Azimuth: 0-360°
   - Elevation: -90 to +90°
   - Distance: 0.1m - 100m
   ```

3. **Head Tracking**:
   ```
   Enable: AirPods Pro/Max
   Response: Real-time position
   ```

### Multidimensional Brainwave Entrainment

Add brainwave entrainment:

1. **Choose Frequency**:
   - Delta (0.5-4 Hz): Deep sleep
   - Theta (4-8 Hz): Meditation
   - Alpha (8-12 Hz): Relaxation
   - Beta (12-30 Hz): Focus
   - Gamma (30-100 Hz): Peak performance

2. **Configure Carrier**:
   ```
   Binaural → Settings:
   Base Frequency: 200 Hz
   Beat Frequency: 10 Hz (Alpha)
   Intensity: 30%
   ```

### Mixing Best Practices

1. **Headroom**: Keep peaks at -6 dB
2. **Bio-Balance**: Let coherence drive dynamics
3. **Spatial Clarity**: Don't overcrowd the field
4. **Frequency Balance**: EQ with bio-awareness
5. **Dynamic Range**: Preserve for streaming

### Mastering

Final polish:

1. **Limiter**: Ceiling at -1 dB (streaming)
2. **LUFS Target**: -14 LUFS (Spotify/Apple Music)
3. **True Peak**: -1 dBTP
4. **Stereo Width**: Check mono compatibility
5. **Reference**: A/B with similar tracks

### Video Export

For visual content:

1. **Resolution Options**:
   - 1080p (YouTube, social)
   - 4K (premium content)

2. **Frame Rate**:

   - H.264 (compatibility)

Export your session data:

```
Export → Bio Data:
✓ Heart Rate Timeline (CSV)
✓ HRV Analysis (JSON)
✓ Coherence Graph (PNG/PDF)
✓ Session Summary (PDF)
✓ MIDI + Bio Context (custom format)
```

### Batch Rendering

For multiple exports:

```
Export → Batch:
1. Add formats to queue
2. Configure naming: {project}_{date}_{format}
3. Set destination folder
4. Start batch render
```

## 7. Intellectual Property Protection

### Copyright Basics

Your music is automatically copyrighted upon creation. However:

1. **Register with Copyright Office**:
   - US: copyright.gov ($45-65)
   - UK: UK Copyright Service
   - EU: Varies by country

2. **Benefits of Registration**:
   - Legal proof of ownership
   - Ability to sue for infringement
   - Statutory damages available
   - Attorney fees recoverable

### Publishing Rights

Two types of rights:

1. **Master Rights**: The recording itself (you own if self-released)
2. **Publishing Rights**: The composition (lyrics, melody, arrangement)

### Registering Your Work

1. **Sound Recording (SR)**: Register the master
2. **Performing Arts (PA)**: Register the composition
3. **Group Registration**: Multiple works together

### PRO Registration

Join a Performing Rights Organization:

- **ASCAP** (USA)
- **BMI** (USA)
- **SESAC** (USA, invite-only)
- **PRS** (UK)
- **SOCAN** (Canada)
- **APRA AMCOS** (Australia)

This ensures you get paid for:
- Radio play
- Streaming
- TV/Film placements
- Live performances
- Background music

### Sync Licensing

For film, TV, ads, games:

1. **Register with Sync Agencies**:
   - Musicbed
   - Artlist
   - Epidemic Sound
   - Songtradr

2. **Negotiate Terms**:
   - Exclusive vs. non-exclusive
   - Territory
   - Duration
   - Media type
   - Fee structure

## 9. Security Best Practices

### Account Security

1. **Strong Passwords**: 16+ characters, unique per site
2. **2FA**: Enable everywhere (prefer hardware keys)
3. **Password Manager**: 1Password, Bitwarden
4. **Recovery Codes**: Store offline, encrypted

### Protecting Your Files

1. **Local Backups**:
   - Time Machine (Mac)
   - External SSD (encrypted)
   - RAID array for large projects

2. **Cloud Backups**:
   - iCloud
   - Backblaze
   - Wasabi (cold storage)

3. **Versioning**:
   - Git for code/presets
   - Splice for sessions
   - Manual version folders

### Secure Collaboration

1. **Share Safely**:
   - Encrypted links (expire after use)
   - Watermarked previews
   - NDA for commercial work

2. **Contracts**: Always have written agreements
3. **Split Sheets**: Document ownership early

### Protecting Biometric Data

Your HRV and heart rate data is sensitive:

1. **On-Device Processing**: Data stays on your device
2. **HealthKit Encryption**: Apple's security model
3. **Export Control**: You decide what leaves device
4. **No Cloud Sync**: Disable if concerned

### API Keys & Credentials

If using streaming/distribution APIs:

1. **Environment Variables**: Never commit keys
2. **Rotate Regularly**: Every 90 days
3. **Least Privilege**: Only needed permissions
4. **Monitor Usage**: Check for anomalies

## 10. Recipe Protection

### What is a "Recipe"?

In Echoelmusic, a "recipe" is your unique combination of:

- Bio-mapping configurations
- AI composition settings
- Effect chains
- Spatial audio presets
- Visual themes
- Session workflows

### Protecting Your Recipes

#### Legal Protection

1. **Trade Secrets**: Keep confidential, use NDAs
2. **Copyright**: Protect preset files, code
3. **Trademark**: Brand your method

#### Technical Protection

1. **Encrypted Presets**:
   ```
   Settings → Export → Preset:
   ✓ Encrypt with password
   ✓ Obfuscate parameters
   ✓ Add watermark ID
   ```

2. **License Keys**:
   - Generate unique keys
   - Tie to device/account
   - Limit installations

3. **Watermarking**:
   - Audio watermarks (inaudible)
   - Visual watermarks
   - Metadata embedding

### Monetizing Recipes

1. **Preset Packs**: Sell on Gumroad, own store
2. **Courses**: Teach the methodology
3. **Licensing**: Let others use with credit
4. **Consulting**: Custom recipe development

Keep detailed records:
