Datenschutz

Stand: Januar 2026

Kurzfassung: Deine Daten bleiben auf deinem Gerät. Wir verkaufen nichts. Du besitzt deine Kreationen.

1. Datenerhebung

Biometrische Daten: Herzfrequenz, HRV werden lokal auf deinem Gerät verarbeitet und nie übertragen.

Audio: Mikrofon-Eingabe für Musikproduktion - bleibt lokal.

2. Keine Server

Wir betreiben keine Server für persönliche Daten. Keine Accounts, keine Cloud.

3. HealthKit

Apple HealthKit Daten werden nie an Dritte weitergegeben, nie für Werbung verwendet, verlassen nie dein Gerät.

4. Deine Rechte (DSGVO)

  • Auskunft, Berichtigung, Löschung
  • Alle Daten lokal → App löschen = Daten weg
