Accessibility Profiles

One-tap profiles optimized for specific needs

Vision

  • Low Vision - Large text, high contrast
  • Blind - VoiceOver with spatial audio
  • Color Blind - 6 safe color palettes

Hearing

  • Deaf - Visual alerts, live captions
  • Hard of Hearing - Audio boost, visual beats

Motor

  • Limited Mobility - Large targets, voice
  • Tremor Support - Stabilized input
  • One-Handed - Reachable controls
  • Hands-Free - Voice + switch nav

Cognitive

  • ADHD - Focus mode
  • Dyslexia - OpenDyslexic font
  • Autism-Friendly - Calm, predictable
  • Memory Support - Context reminders
