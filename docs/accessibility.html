Accessibility Features

  • ✓ VoiceOver – Full screen reader support with descriptive labels
  • ✓ Voice Control – Navigate and create using only your voice
  • ✓ Switch Control – External switch device compatibility
  • ✓ Dynamic Type – Scalable text sizes throughout the app
  • ✓ Dark Mode – Reduced eye strain with dark interface
  • ✓ High Contrast – Enhanced visibility for low vision users
  • ✓ Reduced Motion – Minimize animations for vestibular sensitivity
  • ✓ Color Blind Modes – 6 palettes for different color vision types
  • ✓ Live Captions – Real-time audio transcription
  • ✓ Audio Descriptions – Spoken descriptions of visual content
  • ✓ Haptic Feedback – Tactile responses for all interactions
  • ✓ Eye Tracking – Control with gaze on Vision Pro

Accessibility Profiles

One-tap profiles optimized for specific needs:

Vision

  • Low Vision – Large text, high contrast, bold fonts
  • Blind – VoiceOver with spatial audio descriptions
  • Color Blind – Protanopia, Deuteranopia, Tritanopia modes

Hearing

  • Deaf – Visual alerts, live captions, bass enhancement
  • Hard of Hearing – Audio boost, visual beat indicators

Motor

  • Limited Mobility – Large touch targets, voice control
  • Tremor Support – Stabilized input, longer press times
  • One-Handed – Reachable controls, gesture shortcuts
  • Hands-Free – Complete voice and switch navigation

Cognitive

  • ADHD – Focus mode, reduced distractions
  • Dyslexia – OpenDyslexic font, increased spacing
  • Autism-Friendly – Predictable interface, calm colors
  • Memory Support – Context reminders, clear navigation
