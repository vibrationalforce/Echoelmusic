Common Issues

Wearable not connecting?

Ensure Bluetooth is enabled and your device is nearby. For Apple Watch, grant HealthKit permissions in Settings > Privacy > Health. For other devices, check the companion app is running.

Audio latency issues?

Reduce buffer size in audio settings. On macOS/Windows, use ASIO/Core Audio drivers. Close other audio applications. For best results, use an external audio interface.

DMX not working?

Verify network settings for Art-Net (port 6454). Ensure fixtures are in the correct universe/channel range. Check that USB-DMX interface drivers are installed.

Plugin not showing in DAW?

Rescan plugins in your DAW. Verify the plugin format matches (VST3 for Windows/Mac, AU for Mac only). Check the plugin folder path in DAW preferences.
