1. Einleitung

Willkommen bei Echoelmusic. Diese Datenschutzerklärung erklärt, wie wir Ihre Informationen erfassen, verwenden, offenlegen und schützen, wenn Sie unsere bio-reaktive audio-visuelle Plattform nutzen.

2. Welche Daten wir erfassen

2.1 Biometrische und Gesundheitsdaten

Die App verarbeitet biometrische Daten für bio-reaktive Erlebnisse:

  • Herzfrequenzdaten von Apple Watch, HealthKit oder kompatiblen Wearables
  • Herzratenvariabilität (HRV) für Kohärenzberechnungen
  • Atemfrequenz aus HRV-Analyse
  • Gesichtsausdruckdaten (nur auf dem Gerät verarbeitet)
  • Stimmdaten für Stimme-zu-Visuell-Mapping (auf dem Gerät verarbeitet, nicht gespeichert)
WICHTIG: Alle biometrischen Daten werden AUF DEM GERÄT verarbeitet. Rohe biometrische Daten werden NIE an unsere Server übertragen, es sei denn, Sie aktivieren explizit Cloud-Sync-Funktionen.

3. Ihre Rechte (DSGVO)

  • Recht auf Auskunft: Kopie Ihrer Daten anfordern
  • Recht auf Berichtigung: Falsche Daten korrigieren
  • Recht auf Löschung: Löschung beantragen ("Recht auf Vergessenwerden")
  • Recht auf Datenübertragbarkeit: Daten in maschinenlesbarem Format
  • Recht auf Widerruf: Jederzeit

4. Datensicherheit

  • Verschlüsselung im Ruhezustand: AES-256
  • Verschlüsselung bei Übertragung: TLS 1.3
  • Certificate Pinning: Verhindert Man-in-the-Middle-Angriffe
  • Biometrische Authentifizierung: Face ID, Touch ID, Optic ID
