## ğŸ”§ TROUBLESHOOTING

### **Problem: Samples not appearing**

**Solution:**
```
1. Check file format (WAV/FLAC/AIFF/OGG supported)
2. Check sample rate (44.1kHz or 48kHz recommended)
3. Click "Rescan Library" in Sample Browser
4. Wait for scan to complete
```

### **Problem: Wrong category**

**Solution:**
```
1. Right-click sample
2. "Edit Info"
3. Change category manually
4. Click "Save"
```

### **Problem: No BPM detected**

**Solution:**
```
1. If loop: Name file with BPM (e.g., "Loop_128BPM.wav")
2. Or: Right-click â†’ "Edit Info" â†’ Set BPM manually
3. If one-shot: BPM not needed!
```

### **Problem: Slow scanning**

**Solution:**
```
1. Scan in batches (100-500 samples at a time)
2. Use SSD (not HDD) for faster access
3. Close other apps while scanning
4. Scan runs in background, you can still work!
```
