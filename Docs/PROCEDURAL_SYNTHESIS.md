### 1. Drum Synthesis

#### Kick Drum
- **Pitch Envelope:** Exponentieller Decay von hÃ¶herer zu niedrigerer Frequenz
- **Body:** Sine Wave (Sub-Fundamental)
- **Click:** Kurzer high-frequency Transient
- **Distortion:** Soft-clipping (tanh)

```cpp
// Simplified algorithm
float currentPitch = pitchHz + (pitchStart - pitchHz) * exp(-t * 10.0f);
float sample = sin(currentPitch * TWO_PI * t);
sample += click * sin(phase * 8.0f) * (1.0 - t / 0.005);  // Click transient
sample *= exp(-t / decay);  // Amplitude envelope
sample = tanh(sample * (1.0 + distortion * 3.0));  // Distortion
```

### 2. Bass Synthesis

#### 808 Bass
- **Classic Algorithm:** Sine + 2 Harmonics
- **Saturation:** tanh Drive

```cpp
float sample = sin(phase);                    // Fundamental
sample += sin(phase * 2.0) * 0.3 * tone;     // 2nd harmonic
sample += sin(phase * 3.0) * 0.1 * tone;     // 3rd harmonic
sample = tanh(sample * drive) / drive;        // Saturation
```
